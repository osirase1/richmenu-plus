チャネルアクセストークンの取得方法

  1. LINE Developers にログインします。
  2. 対象のプロバイダーと Messaging API チャネルを開きます。
  3. 「Messaging API設定」または「Messaging API」画面を開きます。
  4. 「チャネルアクセストークン（長期）」を発行または再発行します。
  5. 取得したトークンを、この画面の「チャネルアクセストークン」に貼り付けて保存します。
※ クライアントごとに別のトークンを使用します。別アカウントに反映したい時は、そのアカウントのトークンに切り替えてください。
